# Telco Customer Churn Analysis

## Problem
Predict customer churn to enable proactive retention strategies.

## Key Findings
- 26.6% churn rate
- 88.6% of churned customers on month-to-month contracts
- Average churned customer tenure: 18 months vs 38 months for retained

## Model Performance
- **Random Forest** (selected)
  - Accuracy: 76%
  - Recall: 71% (captures more at-risk customers)
  - ROC-AUC: 0.83

## Recommendations
1. Launch contract conversion campaign
2. Implement "First 90 Days" onboarding program

## Tools
Python, Pandas, Scikit-learn, Matplotlib, Seaborn
